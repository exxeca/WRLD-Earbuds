# Celis_Ezekiel_WRLD_Earbuds  
This repository contains my promotional website project for **MMED 3038 (Electronic Image & Production)** and **MMED 1058 (3D Production)**, created as part of my Level 3 **Interactive Media Design** coursework.  

## Project Name  
**WRLD Earbuds â€“ Promotional Website**  
This project showcases a pair of innovative earbuds and their charging case, modeled and animated in **Cinema 4D**, and presented through an interactive promotional site.  
It demonstrates skills in **3D modeling**, **AR integration**, and **responsive web design**.  

## Features  
- Fully modeled earbuds and charging case (Cinema 4D)  
- Interactive AR view using `<model-viewer>`  
- Hotspots with animated SVG icons  
- Gooey gradient background animated with **GSAP**  
- Responsive, mobile-first layout  
- Clean SASS structure with modular partials  

## Technologies  
- HTML5 / CSS3 / SASS  
- JavaScript (GSAP + Model-Viewer)  
- Cinema 4D  
- Figma (UI design reference)  
- GitHub for version control
